# jetkyll-bootstrap
Bootstrap(v4) theme for Jekyll

## Installation

Add this line to your Jekyll site's Gemfile:

```ruby
gem "jetkyll-bootstrap", git:"https://github.com/chonglou/jetkyll-bootstrap.git"
```

And add this line to your Jekyll site's `_config.yml`:

```yaml
theme: bootstrap
```

And then execute:
```bash
$ bundle install
```
